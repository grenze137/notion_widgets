Photo gallery
---------------------------------
widget that randomly shows one photo from the photos file
---------------------------------
HOW TO RUN:
- if the number of photos in that files is changed, it is necessary to also change the constant totalImages in the index.html
- the images have to be named 1.jpg, 2.jpg etc.
- url to run: https://grenze137.github.io/notion_widgets/photo_gallery/index.html
